# formal-wedding-invitation

A premium formal wedding invitation website designed for relatives, elders, and traditional guests. This edition focuses on essential wedding information, RSVP management, venue details, Islamic blessings, and a refined luxury presentation without personal storytelling elements.

- No romantic photo gallery
- Focused on ceremony information

- RSVP and venue-centered experience
- Suitable for relatives, elders, and family guests

## Features

- **Luxury Design** — Elegant burgundy and gold wedding theme
- **Responsive Layout** — Optimized for desktop, tablet, and mobile devices
- **Wedding Countdown** — Beautiful countdown to the ceremony
- **Interactive Scratch Card** — Elegant invitation reveal experience
- **Wedding Timeline** — Key ceremony milestones and event schedule
- **Event Details** — Clear ceremony and reception information
- **RSVP Management** — Guest attendance confirmation system
- **Venue Directions** — Integrated location and navigation support
- **Guest Wishes** — Digital guestbook for blessings and messages
- **Islamic Blessings** — Meaningful Islamic quotes and duas
- **Premium Animations** — Smooth Framer Motion interactions

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel

## Purpose

This edition was created as a formal alternative to the main wedding website, providing a more traditional and family-oriented invitation experience while maintaining the same premium design quality and modern web technologies.
