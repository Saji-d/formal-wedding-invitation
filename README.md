# formal-wedding-invitation

A formal wedding invitation website designed for relatives and traditional guests, featuring event details, RSVP management, venue directions, and a simplified ceremony-focused experience.

## Related Project

This repository is the formal edition intended for relatives, elders, and traditional guests.

Differences from the main version:
- Simplified experience
- No personal love story timeline
- No romantic gallery
- Focus on wedding information
- Formal presentation
- RSVP and venue focused

Main luxury version:
https://github.com/Saji-d/my-wedding-invitation

## Features

- **Luxury Design:** Burgundy and gold theme with elegant typography.
- **Responsive:** Optimized for both mobile and desktop.
- **Event Details:** Clear information about the wedding ceremony and venue.
- **RSVP Management:** Easy-to-use RSVP form linked to Google Forms.
- **Venue Directions:** Integrated Google Maps for easy navigation.
- **Islamic Blessings:** Meaningful quotes and blessings.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Vanilla CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel
